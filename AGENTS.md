# Contribución en Valinor

## Estilo general
- Prefiere Python con tipeado estático cuando corresponda.
- Evita importar dentro de funciones salvo que sea imprescindible.
- No envuelvas imports en bloques `try/except`.
- Mantén los nombres y textos en español siempre que sea posible.

## Estructura y documentación
- Documenta los comandos de ejecución y despliegue en los README de cada servicio.
- Las rutas de configuración (versiones, variables de entorno) deben indicarse claramente.

## Mensajes y PR
- Resume los cambios en viñetas breves y orientadas a negocio.
- Incluye instrucciones de pruebas ejecutadas.

Estas indicaciones aplican a todo el repositorio salvo que exista un AGENTS.md más específico.
